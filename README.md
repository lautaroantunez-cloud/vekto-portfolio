# VEKTO.dev

Portfolio oficial de VEKTO, estudio digital enfocado en producto, sistemas, presencia web, SEO técnico y automatización.

## Stack
- HTML
- CSS
- JavaScript
- Vercel
- Cloudflare
- Zoho Mail

## Objetivo
Construir una presencia digital sólida para mostrar proyectos, capacidades y estructura de estudio.

## Estructura
- `index.html` → página principal
- `robots.txt` → indexación para buscadores
- `sitemap.xml` → mapa del sitio
- `vercel.json` → configuración de headers y comportamiento en deploy

## Deploy
Este proyecto se publica en Vercel y usa:
- Cloudflare para DNS/CDN
- Vercel para hosting
- Zoho para correo profesional
